Al enviar una petición POST a la ruta http://localhost:3000/registro con el siguiente JSON:

{
"nombre": "Maria",
"mensaje": "Hola comunidad"
}

El servidor recibe los datos mediante req.body y responde con otro objeto JSON confirmando que la información fue recibida correctamente:

{
"estado": "Datos recibidos",
"nombre": "Maria",
"mensaje": "Hola comunidad"
}
