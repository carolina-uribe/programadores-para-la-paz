Al ejecutar node server.js se inicia el servidor web desarrollado con Node.js y Express.

En la terminal aparece el mensaje:
Servidor ejecutándose en puerto 3000

Al visitar la ruta /estado se obtiene una respuesta JSON con información sobre el estado del servicio.

Respuesta obtenida:

{
  "estado": "Servidor funcionando",
  "servicio": "API comunitaria"
}
