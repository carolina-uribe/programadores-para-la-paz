La ruta POST /reporte recibe información enviada por el usuario mediante el cuerpo de la solicitud.

La ruta toma el mensaje enviado y responde con un objeto JSON indicando que el reporte fue recibido correctamente.

Esta ruta podría servir en una aplicación comunitaria para recibir reportes de daños, alertas, problemas de seguridad, quejas o mensajes enviados por los habitantes de una comunidad.
