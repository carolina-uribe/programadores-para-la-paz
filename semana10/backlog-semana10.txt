Backlog (de mayor a menor prioridad)

1. Como ciudadano quiero registrar una incidencia para dejar evidencia del caso.

2. Como coordinador quiero recibir una alerta en Telegram cuando se registre un nuevo reporte.

3. Como miembro del equipo quiero consultar los reportes para hacer seguimiento.

4. Como administrador quiero actualizar el estado de cada caso para mantener la información organizada.
